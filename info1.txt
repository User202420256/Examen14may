Para alojar (o compartir) un repositorio Git en Git en git, siga los siguientes pasos:
Paso 1 : Registar una cuenta en Github
el primer paso para comenzar con el alojamiento en GitHub, es crear una cuenta personal. Visite la pagina de registro oficial para registrarse
Paso 2 Crear un repositorio remoto en Github
Despues de registrase, crear un home (un repositorio)en Github para el repositorio Git que quieras compartir
